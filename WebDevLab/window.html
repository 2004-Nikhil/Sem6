Window Object Functions in JavaScript

1. alert()

The alert() method displays an alert box with a specified message and an OK button.

2. prompt()

The prompt() method displays a dialog box that prompts the user for input.

3. confirm()

The confirm() method displays a dialog box with a specified message, along with OK and Cancel buttons.

4. open()

The open() method opens a new browser window or tab.

5. close()

The close() method closes the current window.
